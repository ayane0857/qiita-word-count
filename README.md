# Qiita 文字数カウンター

> [!NOTE]
> 文字数には誤差があります。
> 空白なしの場合は誤差がすくないですが、空白込みは私の環境では 200 文字程度の誤差がありました。軽い確認程度にお使いください。

Qiita の編集画面で文字数及び行数を確認できるだけの拡張機能です。

[ここ](https://github.com/ayane0857/qiita-word-count/archive/refs/heads/main.zip)からダウンロードできます。

拡張機能のインストールは Chrome の場合はこちらからできます。

`chrome://extensions/`

拡張機能のアイコンをクリックすると文字数をカウントできます
